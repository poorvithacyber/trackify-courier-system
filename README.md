# 🚚 Trackify – Smart Courier Tracking System

## 📌 Project Overview
Trackify is a simple and efficient courier tracking system that allows users to manage and track parcels in real-time.

## 🔧 Tech Stack
- Node.js
- Express.js
- MySQL
- HTML, CSS, JavaScript

## 🚀 Features
- 📦 Add parcel details (Sender, Receiver, Tracking ID)
- 🔄 Update parcel status and location
- 🔍 Track parcel using tracking ID
- 📋 View all parcels
- ❌ Delete parcel
